# Logos fournisseurs

Placez un fichier logo par fournisseur dans ce dossier.

## Fichiers attendus

| Fichier | Fournisseur (clé GHL `supplier`) |
|---------|----------------------------------|
| `sunwing.svg` | `sunwing` |
| `air-canada.png` | `air_canada` |
| `westjet-quebec.svg` | `westjet` / `westjet_quebec` |
| `transat.svg` | `transat` |

## Recommandations

- Format : **PNG** ou **SVG** (fond transparent)
- Largeur cible : **200–400 px**
- Nommez les fichiers exactement comme ci-dessus (config dans `config.js` → `SUPPLIER_LOGOS`)

Pour ajouter un nouveau fournisseur : déposer le logo ici et ajouter une entrée dans `SUPPLIER_LOGOS` dans `config.js`.
